# 🎵 VibeSync - Mood-Based Music Player Dashboard

VibeSync is a modern, high-fidelity music streaming dashboard that matches tailored playlists to the user's current headspace and mood. Moving beyond traditional music streaming layouts, VibeSync features custom vibe metrics, dynamic track analysis, and synchronized lyrics for an immersive listening experience.

🚀 **[Live Demo](https://spotify-clone-snowy-chi.vercel.app/)**

---

## 🌟 Key Features

* **Active Vibe Detection:** Analyzes and tags the user's current music mood (e.g., *Happy Vibes - Bright & Positive Beats*) to curate highly targeted track recommendations.
* **Smart Music Dashboard:** A premium, dark-themed UI featuring search functionalities, track lists, and intuitive playback controls.
* **Metrics Match Analytics:** Visualizes analytical alignment data for every song, breaking down the exact **Vibe Match %** and **Genre Fit %** using sleek progress bars.
* **Now Playing Panel:** Displays expanded album art, localized track metadata, and real-time **Synchronized Lyrics**.
* **Fully Responsive Design:** Crafted with fluid layout structures to ensure a flawless experience across mobile, tablet, and desktop devices.

---

## 🛠️ Tech Stack & Tooling

* **Frontend Library/Framework:** Built with a component-driven architecture for fast rendering and smooth state management.
* **Build Tool:** Powered by **Vite** for blazing-fast development and optimized production asset bundles.
* **Styling Engine:** Styled with **Tailwind CSS** using custom configurations for an ultra-modern design language, smooth hover effects, and strict dark-mode color consistency.
* **PostCSS:** Configured for cross-browser CSS compatibility.
